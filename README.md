# Airbnb Data Analysis Project

## 📊 Project Overview
This project focuses on performing exploratory data analysis (EDA) on Airbnb listing data to uncover insights related to room types and neighborhood trends.

## 🛠 Tools & Technologies
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn

## 🧹 Data Cleaning
- Checked and handled missing values
- Removed duplicate records

## 📈 Key Findings
- Entire home/apt was the most dominant room type (50k+ listings)
- Manhattan had the highest number of listings (40k+)
